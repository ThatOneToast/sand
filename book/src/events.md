# Events

This legacy source path is retained for existing repository links. The canonical,
navigation-linked mdBook chapter is [Manual: Events](manual/events.md).

That chapter documents the current `AdvancementEvent`/`SandEvent` split,
`Event<T>` runtime context, typed custom dispatch, generic event definitions,
implemented single- and multi-parent same-cycle composition, persistent
conditions, bounded cross-tick correlation, and the later
context-propagation/participant phases that are not yet APIs.

Version-aware advancement-trigger coverage and evidence remain documented in
[Advancement Triggers](manual/advancement-triggers.md) and the
[Event Trigger Matrix](reference/event-trigger-matrix.md).
