# Command Builders

Common builders: typed generated `cmd::*` commands; `sand_commands::{tp_vec3, tp_with_rotation, summon_at, summon_at_with_nbt}`; `cmd::effect_give`; `Damage`; display/text/sound/particle builders; inventory and movement systems. Builders serialize to commands; inspect `to_string()` in a test when verifying output.
