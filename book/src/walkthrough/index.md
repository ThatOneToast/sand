# Walkthrough Overview

This walkthrough builds a compact `arcane` datapack: a mana score, a magic wand, a shockwave shield, an altar, and a damage-reactive effect. Chapters build on one another, but the theme is only a vehicle for learning Sand.

Each step links to a focused manual page. Keep the project open while reading: build with `cargo run -p sand -- build`, install the output in a world, then `/reload` after changes.
