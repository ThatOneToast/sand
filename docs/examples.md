# Examples

Examples should be copyable into a Sand project and should prefer:

- `use sand_core::prelude::*`
- Typed command builders
- Typed state and conditions
- Typed dialogs, items, text, storage, and component builders

Current example categories:

- Basic typed function/load/tick flow
- State and conditions
- Dialogs
- Storage and NBT
- Datapack components
- Interop escape hatches

Raw examples belong only in the interop escape-hatch example.
